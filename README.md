# Gektor Vector Tracer

Chrome DevTools panel extension for raster-to-vector tracing using ImageTracer.

## Install
1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Load unpacked extension and select this folder.

## Usage
- Load image from file or URL.
- Click `Trace` to generate vector.
- Use panel controls to save SVG/EPS or clear.

## Dev
- Start local server: `python -m http.server 8000`.
- Open `panel.html` to test.
